## 🔧 Common Issues and Fixes

**❌ "Injection Failed" error on Windows**
> Launch `FluxusZ.exe` as **Administrator**. If the problem persists, add the Fluxus Z installation folder to your antivirus exclusion list — Windows Defender and most third-party AV tools flag executor binaries due to their memory-writing behavior, which is a false positive.

**❌ Android APK installs but crashes on launch**
> Ensure your device runs Android 8.0 or higher and that you're using the current **Fluxus Z APK 2026** from this repository. Delete any older APK versions before installing the new one — conflicting installs cause launch crashes.

**❌ iOS shows "Unable to Verify App" or "Untrusted Developer"**
> Go to **Settings → General → VPN & Device Management**, find the Fluxus Z entry, and tap **Trust**. This must be repeated every 7 days with a free Apple ID. If the option is greyed out, revoke and re-sideload the IPA.

**❌ Script runs without errors but nothing happens in-game**
> The script may be targeting an outdated game version or a game you're not currently in. Check the Script Hub for an updated version of the same script. Also confirm you are fully loaded into the game world — executing from the lobby screen causes silent failures on many scripts.

**❌ Auto-Execute doesn't fire on game join**
> Open **Settings** inside Fluxus Z and verify that "Auto-Execute on Join" is toggled ON. Also confirm that your saved script slots contain valid scripts — the feature silently skips empty slots without showing an error.

**❌ Executor stops working after a Roblox update**
> This is expected behavior after major Roblox client patches. The Fluxus Z auto-update engine handles most routine patches silently, but significant Roblox updates may require a manual build update. Check this repository's releases page for the **Fluxus Z executor latest version** — patches are typically deployed within 24–48 hours.

**❌ Drawing API visuals not rendering on mobile**
> This usually indicates insufficient available RAM. Close all background apps, restart Roblox and Fluxus Z, then re-inject. On devices with exactly 4 GB RAM, avoid running Drawing API scripts alongside other heavy GUIs simultaneously.
